# CellSim

Simple Monte Carlo optical model for 3D cell clusters

## Features

*   **Shape Generator:** Generates simple geometries.
*   **Volume Generator:** Generates optical properties of a volume based on geometries.
*   **Illumination Generator:** Sets up illumination of volume.
*   **Optical Simulation:** Simulates photon-volume interactions.
*   **Virtual Microscope:** Simulates light capture and imaging.
